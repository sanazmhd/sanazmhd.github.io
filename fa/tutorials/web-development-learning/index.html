🎯 آموزش ساخت یک پروژه کامل: از بک‌اند تا فرانت‌اند با Flask، React، MySQL و Docker

مهٔ 6, 2025 · 2 دقیقه

اگر تازه وارد دنیای برنامه‌نویسی شده‌اید و می‌خواهید بفهمید که فرانت‌اند، بک‌اند، دیتابیس و داکر دقیقاً چه هستند و چرا مهم‌اند، این بلاگ احتمالا براتون مفید باشه.

🧩 فرانت‌اند (Frontend) چیست و چرا نیاز داریم؟#

فرانت‌اند همان چیزی است که کاربر می‌بیند: ظاهر سایت یا اپلیکیشن.

مثلاً وقتی به یک فرم در سایت می‌روید و اسمتان را وارد می‌کنید و روی دکمه‌ی “ارسال” کلیک می‌کنید — این بخش توسط فرانت‌اند ساخته شده.

در این پروژه، ما از React برای ساخت فرانت‌اند استفاده کردیم چون:

  • طراحی زیبا و پویا می‌خواهیم.
  • کاربران راحت بتوانند اطلاعات وارد کنند.
  • بدون رفرش شدن صفحه، اطلاعات نمایش داده شود.

⚙️ بک‌اند (Backend) چیست و چه کاری انجام می‌دهد؟#

بک‌اند بخشی از برنامه است که در پشت صحنه کار می‌کند:

  • داده‌ها را ذخیره می‌کند.
  • بررسی می‌کند درخواست کاربر درست است یا نه.
  • اطلاعات را از پایگاه‌داده می‌گیرد و به فرانت‌اند می‌فرستد.

ما در این پروژه از Flask (یک فریم‌ورک پایتون) برای بک‌اند استفاده کردیم چون:

  • سبک و ساده است.
  • به راحتی با دیتابیس ارتباط برقرار می‌کند.
  • خیلی مناسب برای پروژه‌های کوچک و متوسط است.

🗄 دیتابیس (Database) چیست و چرا استفاده کردیم؟#

دیتابیس جایی است که اطلاعات ذخیره می‌شوند.

ما در این پروژه نام کاربران را ذخیره می‌کنیم. برای این کار از دیتابیس MySQL استفاده کردیم.

مثال: اگر کاربری اسم “امیر” را وارد کند، در جدول دیتابیس یک ردیف جدید اضافه می‌شود با مقدار Amir.

🐳 داکر (Docker) چیست و به چه دردی می‌خورد؟#

فرض کنید پروژه‌تان روی کامپیوتر شما اجرا می‌شود، ولی وقتی روی کامپیوتر دوست‌تان می‌گذارید، کار نمی‌کند! 😞

Docker این مشکل را حل می‌کند.

داکر یک محیط مجازی می‌سازد (به نام کانتینر) که همه‌چیز داخلش مشخص و ثابت است: پایتون، Flask، MySQL، همه چیز.

مزیت داکر:

  • دیگر مهم نیست روی چه کامپیوتری اجرا می‌کنید.
  • برنامه‌تان همیشه دقیقاً به یک شکل کار می‌کند.

🛠 مراحل کلی پروژه#

  1. ساخت بک‌اند با Flask
    مسیرهایی تعریف کردیم برای:

    • نمایش فرم
    • ذخیره‌سازی نام‌ها
    • نمایش لیست کاربران

  2. ساخت دیتابیس با MySQL
    دیتابیسی به نام flask_app ساختیم و اطلاعات را داخل جدول User ذخیره کردیم.

  3. ساخت فرانت‌اند با React
    فرمی ساختیم که به Flask درخواست می‌فرستد و لیست کاربران را نمایش می‌دهد.

  4. اجرای پروژه با Docker
    با یک فایل Dockerfile و دستورات:

    docker build -t flaskapp .
docker run -p 5000:5000 flaskapp
