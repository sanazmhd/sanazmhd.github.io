### 🌐 Web Development Learning!

🔍 Overview

This tutorial will guide you step-by-step through building a full-stack web application using:

    React.js for the frontend (user interface)

    Flask for the backend (server and API)

    MongoDB for the database (data storage)

    Docker to run each part in its own container

You’ll learn how to build, connect, and run each component separately — just like in a professional web development environment.
🏗️ What You Will Build

A simple web app where users can:

    Enter their name in a textbox.

    Click a button to submit the name.

    The name is sent to the Flask backend.

    Flask saves it in a MongoDB database.

    (Optional) Display the list of saved names on the page.

⚙️ Technologies Used
Layer	                    Tool	                Purpose
Frontend        	            React.js	User        Interface (UI)
Backend	                    Flask (Python)	    Handles logic & API requests
Database	                    MongoDB	            Stores submitted data (names)
Containerization 	        Docker & Docker     Compose	Isolate and manage services

📦 App Architecture

All components are separated into individual containers:

    React (Frontend)

    Flask (Backend)

    MongoDB (Database)
    
    🎯 Learning Goals

    Understand frontend/backend/database roles

    Create REST APIs with Flask

    Communicate between React and Flask using HTTP

    Save and retrieve data with MongoDB

    Use Docker and Docker Compose to run everything cleanly
