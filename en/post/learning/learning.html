ASP.NET with Dofactory

The Dofactory .NET 6 framework provides a comprehensive set of tools and examples to help .NET Core developers build robust applications quickly. It focuses on leveraging design patterns, clean architectures, and low-code solutions.

Introduction

Dofactory .NET 6 is a rapid application development framework tailored for .NET Core developers. It assumes familiarity with C# and offers numerous reference applications to demonstrate real-world coding techniques and design patterns. The framework supports .NET Core 6 and higher, incorporating the latest language features such as nullable reference types and minimal hosting models.

Setup and Installation

The framework is distributed as a zip file. After unzipping, it's recommended to install it in a local directory (e.g., C:\Users\%username%\Documents\Dofactory.NET 6\*.*). The package includes multiple editions: Standard, Plus, and Pro, each adding more advanced features and reference applications.

To run the applications, Visual Studio 2022 or later is required. Ensure that you have the Web Development and .NET Desktop Development workloads installed. The applications use LocalDb by default but can be configured to use SQL Server if preferred.

Key Features and Editions

  • Standard Edition: Focuses on design patterns like Gang of Four and Enterprise Architecture. Includes applications like Analytics and Art Shop.
  • Plus Edition: Introduces CQRS and SaaS patterns, along with a modern multi-tenant SaaS application and the Spark code generator.
  • Pro Edition: Completes the package with chapters on Ultra-Clean Architecture and the 33-Day App Factory, including the 33-Day CRM application.

Learning Path

For a comprehensive understanding, start with Chapters 2 (Design Excellence) and 3 (Code Excellence). Students should follow all chapters sequentially, while mid-level developers can start at Chapter 6, and experienced architects might focus on Chapters 9 and onward.

Usage and Licensing

The Dofactory .NET 6 framework can be used in personal and commercial applications. However, it cannot be used to create competing frameworks or libraries. The package comes with a single-developer license, and site licenses are available for multiple users.
